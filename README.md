# one-toc
为react官方文档添加文档目录的chrome插件

- header 偏移
  - 默认偏移量：60px
  - header高度覆盖默认
- 锚点控制
  - id、name
  - 点击事件控制
- 解析错误：非内容区域的h2、h3被解析为锚点
